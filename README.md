This project was built as **Day 25 of the 100 Days of Python Code** course to understand how to read and work with CSV files in Python. Pandas was used to efficiently load, analyze, and access tabular data containing U.S. state names and their coordinates.

On **Day 26**, the project was improved using list and dictionary comprehension with conditionals. Iterating over Pandas DataFrames using list comprehension helped make the logic more concise and expressive, reinforcing Pythonic data handling techniques.
